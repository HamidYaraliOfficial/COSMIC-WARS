# COSMIC WARS

## 🚀 English

### Overview
COSMIC WARS is a modern 2D sci‑fi artillery game built with Python and PyQt6. Choose a faction, launch advanced missiles, manage ammunition, battle AI-controlled enemies, and dominate the galaxy through tactical warfare.

### Features
- Multiple playable factions
- Different missile classes and nuclear weapons
- Dynamic wind system
- AI-controlled opponents
- Score tracking system
- Particle explosion effects
- Dark and Light themes
- Multi-language support (English, Persian, Chinese)
- Modern PyQt6 graphical interface

### Requirements
- Python 3.10 or newer
- PyQt6

### Installation

#### Install Required Libraries
```bash
pip install PyQt6
```

#### Run the Game
```bash
python "COSMIC WARSCOSMIC WARS.py"
```

### Gameplay
1. Select a faction.
2. Choose a missile type.
3. Adjust angle and power.
4. Launch missiles toward enemy bases.
5. Destroy all enemy factions before your base is destroyed.

# 中文

## 🌌 项目简介

COSMIC WARS 是一款基于 Python 和 PyQt6 开发的二维科幻战略战争游戏。玩家可以选择不同阵营，发射各种导弹和核武器，与人工智能敌人展开激烈战斗。

### 功能特点
- 多个可选阵营
- 多种导弹与核武器
- 动态风力系统
- AI 敌人系统
- 分数统计
- 粒子爆炸效果
- 深色与浅色主题
- 多语言支持
- 现代化图形界面

### 系统要求
- Python 3.10 或更高版本
- PyQt6

### 安装

#### 安装所需库
```bash
pip install PyQt6
```

#### 运行游戏
```bash
python "COSMIC WARSCOSMIC WARS.py"
```

### 游戏玩法
1. 选择一个阵营。
2. 选择导弹类型。
3. 调整发射角度和力度。
4. 向敌方基地发射导弹。
5. 在基地被摧毁前消灭所有敌人。
